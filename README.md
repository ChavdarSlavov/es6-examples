# ES6 training build

## Requements
 - nodejs 4.4
 
## First run
You need to install some additional node packages
```sh
$ npm install -g webpack
$ npm install -g webpack-dev-server
$ npm install
```

## Commands 

Start environment - http://localhost:3000 
```sh
$ npm start
``` 


Build files in `dist` folder:
```sh
$ npm run build
```
